
# Fina Arte Sorvetes – Management System

Sistema desenvolvido para apoiar a gestão da sorveteria Fina Arte Sorvetes, criado durante um projeto de extensão universitária com cliente real.

O objetivo foi desenvolver uma solução digital para facilitar o gerenciamento de produtos e operações do negócio.

O sistema não está mais em produção e foi mantido apenas para fins acadêmicos e de portfólio.

## Tecnologias Utilizadas

Backend: Java, Spring Boot, MongoDB

Frontend: React, JavaScript, HTML / CSS

Swagger (documentação da API)
